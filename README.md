# Trešnje Kljajo 🍒

Dobrodošli na službenu web stranicu **Trešnje Kljajo** – domaće, svježe trešnje iz Vojna kod Mostara.

---

## 📌 Opis

Ovo je profesionalna web stranica za prodaju domaćih trešanja. Stranica je dizajnirana da odmah privuče kupce:

* Velika hero slika na naslovnoj
* Automatska galerija svih slika
* Lightbox prikaz slika
* WhatsApp/Viber dugmad za brzu narudžbu
* Informacije o dostavi (Mostar i okolica)
* Animacije za moderne premium efekte

---

## 📂 Struktura projekta

```
Tresnje-Kljajo/
├─ index.html        # Glavna HTML stranica
├─ css/              # CSS datoteke (ako ih dodaš kasnije)
├─ js/               # JavaScript datoteke (ako ih dodaš kasnije)
├─ img1.jpg          # Primjer slike za galeriju
├─ img2.jpg          # Primjer slike za galeriju
├─ hero.jpg          # Hero slika
├─ README.md         # Ova datoteka
```

---

## 🚀 Instalacija i pokretanje

Ako želiš lokalno pregledati stranicu:

1. Kloniraj repozitorij:

```bash
git clone git@github.com:dkljajo/tresnje-kljajo.git
```

2. Uđi u folder:

```bash
cd tresnje-kljajo
```

3. Otvori `index.html` u pregledniku ili koristi lokalni server (npr. VS Code Live Server).

---

## 🌐 Objavljivanje na GitHub Pages

1. Pushaj sve datoteke u main branch:

```bash
git add .
git commit -m "Prva verzija web stranice"
git push -u origin main
```

2. GitHub Pages Settings:

* Branch: `main`
* Folder: `/ (root)`
* Klikni **Save**

Link za pregled stranice: `https://dkljajo.github.io/tresnje-kljajo/`

> Napomena: **index.html mora biti u root folderu**.

---

## 📸 Dodavanje slika

* Hero slika: `hero.jpg`
* Galerija: sve slike navedene u `galleryFolder` nizu u `index.html`
* Klikom na slike se otvara lightbox.
